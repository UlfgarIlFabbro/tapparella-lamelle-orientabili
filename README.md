# Tapparella Lamelle Orientabili

Integrazione Home Assistant per tapparelle Cherubini con lamelle orientabili.

## Requisiti

- Isolatore Cherubini A510052 o compatibili
- Motore con fine corsa elettronico
- Shelly in modalità tapparella
- Shelly configurato in Detached

## Funzioni

- Apertura tapparella
- Chiusura breve (Shelly impulse)
- Lamelle (tilt motore)

## Installazione

Aggiungere repository in HACS e installare integrazione.
